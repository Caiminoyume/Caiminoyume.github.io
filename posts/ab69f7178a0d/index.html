为 openwrt 编译 nginx

本文最后更新于：2024年1月19日 下午

起因

由于 openwrt 使用 musl-gcc，无法运行 gcc 编译的程序，需要额外编译

并且 openwrt 官方编译的 ipk 自由度不高，需要自己编译功能

准备工作

参见上篇：编译 Nginx 并开启 HTTP/3

编译 musl-gcc

  1. musl 源码，解压 tar.gz，目录：$WORKER/musl
  2. cd ./musl ./configure make sudo make install
  3. x86_64-linux-musl-gcc -v确认安装完毕
  4. export CC=/usr/bin/x86_64-linux-musl-gcc，将编译器设为 musl-gcc

开始编译

  1. 进入 nginx 目录cd ./nginx
  2. 配置：
  3. make，编译结果会存于$WORKER/nginx/objs
  4. 安装（不必须）：sudo make install
